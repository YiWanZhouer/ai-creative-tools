# 🛠️ AI 创意工具箱

AI 驱动的综艺策划辅助工具。选好创意元素 → 自动生成结构化方案。

## 工具

### 🎬 方案骨架生成器

从 8 个维度、58 个创意元素中自由组合，自动生成：

- **方案简介**：一句话说清这个综艺讲什么
- **观众旅程**：刚入坑 → 追到中段 → 到大结局，每个阶段的观看体验
- **核心看点**：元素组合的独特化学反应
- **因果推导链**：为什么这个组合能成立（15 条预置推导链）
- **边界条件分析**：在什么条件下会翻车、怎么修
- **中国市场适配**：政策红线、观众偏好、本土化建议
- **AI 润色 / 脑暴 / 诊断 / 商业分析**：一键调用 DeepSeek V4 Pro

### ✨ 创意方案生成器

输入一句话创意 → 一键生成完整方案文档，含观众旅程、元素拆解、市场适配。

## 怎么用

1. 双击 HTML 文件在浏览器打开
2. 点击 ⚙️ → 填入 DeepSeek API Key（[获取 Key](https://platform.deepseek.com/api_keys)）
3. 勾选创意元素 → 点击"生成方案"
4. 可选：点击 🤖 润色 / 🧠 脑暴 / 🔍 诊断 / 💼 商业 获得 AI 深度分析

> Key 保存在浏览器 localStorage，下次打开无需重填。
