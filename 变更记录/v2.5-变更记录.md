# 创意方案生成器 · v2.4 → v2.5

**日期**：2026-06-19
**审查者**：AI工具审查者（agent-718811e3）—— 方案审计 4 FLAG（已闭合），产出审计条件PASS（全验证通过）

## 新增功能

### 方案诊断模式
- 页面新增模式切换 tabs：✨ 生成方案 / 📋 提案诊断
- 诊断模式：隐藏四维矩阵 + 项目信息 + 颗粒度控件，显示 textarea（粘贴已有方案）
- `diagnoseProposal()`：50 字门槛验证 → DeepSeek API → 四维矩阵结构化诊断
- 系统提示词：6 段结构化输出（维度覆盖/核心优势/关键风险/类型片偏差/调整建议/一句话总结）
- 诊断结果缓存：切换回生成模式后切回，诊断结果不丢失
- API 进行中禁用 tab 切换，防止误操作
- 失败时清除缓存，按钮/tab 正常恢复

## 技术细节
- 新增 state：currentMode / isDiagnosing / cachedDiagnosis
- 新增 switchMode()：tab 切换 + 区域显隐 + 缓存恢复 + API 锁定
- 新增 CSS：mode-tabs / mode-tab / diagnose-area / diagnose-field
- 新增 buildDiagnoseSystemPrompt()：6 段结构化诊断指令
- 诊断结果复用现有 renderMarkdown() + editPanel 编辑流程
- 输出 footer 含复制 + 编辑按钮
