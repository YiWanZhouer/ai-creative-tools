# 方案骨架生成器 · v1.5 → v1.6

**日期**：2026-06-18
**审查者**：AI工具审查者（agent-718811e3）

## 新增功能

### 元素优先级引擎（L1→L2→L3）
- 八维度按三层分层：L1 发端层（叙事引擎+时间结构）→ L2 丰富层（空间/筛选/人物/观看）→ L3 审视层（出口/真实性）
- L2/L3 默认锁定，逐层解锁：L1 填完 → L2 解锁；L2 三选三 → L3 解锁
- **重锁机制**：取消 L1 选择 → L2/L3 自动重锁
- L1 修改保护：L2/L3 有选择时修改 L1 → 确认弹窗 → 确认后清空下游 + 回溯提示条
- 预设方案/随机生成自动解锁所有层

### 类型片分类体系
- 6 种预设类型片：选秀/竞技类、恋综/情感类、观察类、生存/挑战类、群居实验类、旅行/公路类
- 每种类型片有专属 L1 hint 提示和建议元素高亮
- 切换类型片不清空已有选择
