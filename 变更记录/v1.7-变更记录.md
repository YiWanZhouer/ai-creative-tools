# 方案骨架生成器 · v1.6 → v1.7

**日期**：2026-06-18
**审查者**：AI工具审查者（agent-718811e3）

## 新增功能

### 逆规律案例库
- 10 个预制案例，覆盖全部 6 种类型片
- 案例数据结构：名称/类型片/打破的规律/特殊条件/爆款归因/可复用条件/不可复用条件
- 自动匹配：根据当前方案激活的因果链 + 类型片匹配度评分
- 最多展示 3 个最相关案例

### 输出面板增强
- 因果推导链和边界条件之间新增「🔮 逆规律案例」section
- 案例卡片展示：类型片 badge + 打破的规律 + 当时条件 + 归因 + 可复用/不可复用

### AI 脑暴/诊断增强
- 脑暴 prompt 注入匹配案例上下文
- 诊断 prompt 注入黑马模式参考

## Bug 修复
- `generateHighlights` 遍历 `Object.keys(state)` 时碰到新增的 `activeGenre: null` → forEach 报错。已加 skip 逻辑。
